# LeadMaster — Glosario de dominio

Glosario canónico del dominio de prospección LeadMaster. Solo lenguaje de dominio:
no contiene detalles de implementación, SQL ni rutas de archivo.

## Language

**Prospecto capturado**:
Una empresa/landing registrada por primera vez al aparecer en una búsqueda de prospección. Se identifica por su landing, y conserva la primera keyword base, la localidad de búsqueda y la query completa con que apareció esa primera vez.
_Avoid_: registro, captura, lead crudo

**Primera aparición**:
El primer momento en que un landing es detectado en una búsqueda. Define la atribución (keyword base, localidad, geo) que queda fijada para ese prospecto; las apariciones posteriores del mismo landing no la modifican.
_Avoid_: alta, primer registro

**Prospecto validado**:
Prospecto capturado que fue evaluado y marcado como válido.
_Avoid_: prospecto ok, aprobado

**Empresa**:
Entidad jurídica detectada en el landing o en el texto extraído.
_Avoid_: compañía, firma

**Lead**:
Prospecto calificado listo para contacto comercial.
_Avoid_: contacto comercial, prospecto caliente

**Interés manifiesto**:
Evidencia de intención de compra en el texto extraído del landing.
_Avoid_: intención, señal de compra

**Cliente LeadMaster**:
Empresa que contrató LeadMaster. No confundir con prospecto.
_Avoid_: cliente, cuenta

**Vertical**:
Sector industrial objetivo de prospección (ej.: seguros, reformas, salud).
_Avoid_: rubro, industria, nicho

**Muestra**:
Conjunto acotado de prospectos usado para validación.
_Avoid_: batch, lote de prueba

**Contacto**:
Email, teléfono o WhatsApp extraído del landing.
_Avoid_: dato de contacto, medio

**Oportunidad comercial**:
Lead calificado con contacto y scoring positivo.
_Avoid_: deal, venta potencial

## Términos de búsqueda territorial

**Keyword base**:
El término de negocio de una búsqueda, sin componente geográfico (ej.: "broker de seguros").
_Avoid_: palabra clave (ambigua: puede referir a la query completa)

**Localidad de búsqueda**:
El lugar usado como modificador geográfico de una búsqueda (ej.: "Cordoba"). Es una ubicación para armar la query, no el domicilio del prospecto.
_Avoid_: ubicación, domicilio, zona

**Modificador geográfico**:
El texto de localidad que se combina con la keyword base para formar una búsqueda territorial.
_Avoid_: geo, sufijo geográfico

**Query completa**:
La combinación de keyword base más localidad de búsqueda enviada al buscador (ej.: "broker de seguros Cordoba"). Una query completa no es un prospecto: solo hay prospecto cuando existe un landing capturado.
_Avoid_: búsqueda, query google, término de búsqueda
